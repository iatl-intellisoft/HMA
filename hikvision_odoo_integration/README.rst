Hikvision Biometric Device Integration
===============================
* This Cybrosys's module integrates Odoo attendance with hikvison biometric device attendance.

Configuration
============
- This integration is  applicable for the the Hikvision Devices.

Compatible Devices
----------------
This module support with the following machines

* DS-K1T343MFWX (Hikvision)
